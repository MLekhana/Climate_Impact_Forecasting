# 🌍 Climate Impact Forecasting

This project is a simple web application developed using Streamlit that predicts future climate conditions based on user inputs such as city, date, duration, and selected parameter (temperature, rainfall, or humidity). It uses a basic machine learning model to generate predictions and provides useful suggestions to help users plan activities like travel or daily tasks. The application also displays graphical representations of temperature, rainfall, and humidity trends for better understanding.

The main objective of this project is to create awareness about climate change and show how technology can be used to analyze and predict environmental conditions. It offers a clean and user-friendly interface where users can easily interact and get quick insights. The project is suitable for academic purposes and demonstrates the use of data visualization, user input handling, and basic predictive analysis in a practical way.

Overall, this application highlights how simple tools and models can be combined to build an interactive system that helps in understanding climate patterns and making better decisions.
